# Simple Origin - Brave Debloat 0.6.0

## Highlights

- Simple Origin now requires administrator rights on launch.
- This avoids unpredictable registry ACL failures when Brave policy keys under HKCU or HKLM were created by elevated processes.
- The README launch instructions now tell users to open PowerShell as Administrator first.

## Recommended launch

Open PowerShell as Administrator, then run:

```powershell
irm https://raw.githubusercontent.com/unmatched785/SimpleOrigin-Brave-Debloat/refs/tags/0.6.0/SimpleOrigin.ps1|iex
```
